📖 Complete Setup Guide

From download to fully running bot in under 5 minutes.

1

Download & Launch

Download FlyffSmartFS.exe and run it. Windows Defender may show a warning — click "More info" → "Run anyway" (see Defender guide below).

The license dialog appears: click "Start Trial" for 30 minutes free with all features, or enter your license key.

2

Open Browsers

Recommended: in the GUI, click "Open FS Browser" and "Open Main Browser". This launches Chrome/Edge in debug mode and opens Flyff Universe directly.

Already have Flyff open? Since v1.0.22, SmartFS also detects already-running Flyff windows — including installed apps (PWAs). On the first scan or start, the bot offers a one-click restart with debug port. Your login stays intact.

3

Log into Flyff

In both browser windows: navigate to Flyff Universe, log in, and press F11 for Fullscreen (1920x1080 required).

Make sure FS and Main are in the same party. On your FS browser:

  • Open the party window with HP bars on the LEFT side
  • Position names centered and readable
  • Open the character info window (press T)
  • Hide HP numbers in char info (only show the bar)
4

Enter Character Names

In the GUI "Setup" tab, type your FS Name and Main Name exactly as they appear in-game.

⚠️ Case-sensitive! "MyChar" ≠ "mychar" ≠ "MYCHAR". No trailing spaces.

5

Scan Names (10 seconds)

Make sure your FS is at 100% HP, then click "SCAN NAMES":

  • Phase 1: Click on Main's name in the party list → green rectangle
  • Phase 2: Click on FS name → blue rectangle
  • Phase 3: View switches to char info → click your character name → orange rectangle

Done! The bot has learned your UI layout. Re-scan anytime if needed.

6

Change Deselect Target Key

In Flyff's game settings (ESC → Settings → Controls): change "Deselect Target" from ESC to ^ (key left of 1).

⚠️ ESC is bugged in Flyff Universe — it opens menus and interferes with the bot. The ^ key works cleanly.

7

Configure & Start

In the "Skills Main" tab:

  • Enable Heal or Heal Rain (never both!)
  • Enable the buffs you want and enter matching Flyff hotkeys
  • Enable Auto Rebuff and Auto Follow

Hit START. Your FS is now fully autonomous.

Killswitch (ALT) stops everything instantly — configurable in the Setup tab.

⚠️ Windows Defender Guide

SmartFS is not code-signed yet — Defender may flag it as suspicious. This is a false positive.

When launching the EXE:

  1. Windows shows "Windows protected your PC"
  2. Click "More info"
  3. Click "Run anyway"

Add to Defender exclusions (recommended):

  1. Open Windows Security (search in Start menu)
  2. Go to Virus & threat protection
  3. Click Manage settings under "Virus & threat protection settings"
  4. Scroll to Exclusions → click Add or remove exclusions
  5. Click Add an exclusion → Folder
  6. Select the folder where FlyffSmartFS.exe is located
  7. Done! Defender will no longer interfere.

If Defender already deleted the EXE:

  1. Open Windows Security
  2. Go to Protection history
  3. Find the SmartFS entry → click Actions → Allow on device
  4. Re-download the EXE

❓ Frequently Asked Questions

Is there a free trial?

Yes! 30-minute free trial with all Premium features unlocked. No key required — just download and start. Trial time only counts while the bot is actually running (START pressed), not during setup.

What resolution is supported?

1920x1080 Fullscreen is the main target. 1440p and 4K are also supported since v1.0.2 — the scan dialog and bar measurements scale proportionally.

Can I use my PC while it runs?

Yes, but Flyff must stay in the foreground. ALT+TAB away or minimizing the Flyff browser triggers Chromium's WebGL background throttling — the bot sees stale pixels and reacts with delay. Tip: use a second monitor or position the bot GUI next to Flyff.

Do I need both accounts on one PC?

Yes. Both Flyff browsers run on the same machine. Multi-PC setups are not supported.

Chrome or Edge?

Both work, including Flyff as an installed app (PWA) since v1.0.22. The bot auto-detects already-running instances and offers a one-click relaunch with debug port.

Can I use Flyff as an installed app?

Yes. Since v1.0.22 the bot supports PWA (Edge/Chrome "Install as app"). Limitation: if BOTH chars are PWAs, Chromium shares one process — use mixed mode (one PWA + one regular tab) or two different browsers.

Is it detectable?

SmartFS uses proprietary screen reading and input simulation. No memory injection, no DLL hooking. Designed to be invisible to anti-cheat. Botting violates Flyff ToS — use at your own risk.

My key says "already activated"?

Keys are bound to one PC. If you reinstalled Windows or changed hardware, join our Discord and open a support ticket for a key reset.

Windows Defender blocks the EXE?

The EXE is not code-signed yet — false positive. See the Defender Guide above for step-by-step instructions.

What about Holy Cross?

HC is in Alpha. It uses a custom AI model (~200 MB one-time download). Works best with FS close to Main, camera zoomed out, monsters that live 5+ seconds.

How fast are bug fixes?

Usually within hours. Fixes deploy via auto-update — no manual download needed. Report issues on Discord or via PM.

Does it work on private servers?

SmartFS is built for Flyff Universe (browser version). Private servers with different clients are not supported.

Where can I get support?

Join our Discord server — we have English and German support channels, setup guides, FAQ, and a ticket system.
